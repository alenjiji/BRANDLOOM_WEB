# BrandLoom

Website for BrandLoom, a premium brand identity and design studio.

## Stack

- **Next.js 16** with App Router
- **React 19** — fully server-rendered (no client components)
- **Tailwind CSS v4** — utility-first, no config file
- **TypeScript 5**


## Project Structure

```
app/
  layout.tsx      # root layout, font loading
  page.tsx        # home page
  globals.css     # design tokens, typography utilities
components/
  HeroSection.tsx
  AboutSection.tsx
public/hero/      # brand SVG assets
```
