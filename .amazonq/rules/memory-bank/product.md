# Product: BrandLoom Website

## Purpose & Value Proposition
BrandLoom is a premium brand identity and design agency website. It presents the studio's services and aesthetic through a high-fashion "Digital Atelier" editorial design language — communicating craft, precision, and luxury to prospective clients.

## Key Features & Capabilities
- **Editorial Hero Section** — Full-viewport hero with serif display typography, asymmetric layout, and SVG brand assets (logo, loom, pc illustrations)
- **About Section** — Studio narrative and positioning content
- **Design System** — Custom CSS utility classes (`hero-title`, `hero-italic`, `section-title`, `small-label`) implementing the brand's typographic voice
- **Responsive Typography** — Fluid `clamp()`-based type scaling across all breakpoints
- **Dark Mode Support** — CSS custom property swap via `prefers-color-scheme`

## Target Users & Use Cases
- **Prospective clients** evaluating BrandLoom for brand identity, web design, or creative direction projects
- **Internal team** using the site as a living portfolio and brand reference
- **Stakeholders** reviewing the studio's positioning and aesthetic direction

## Brand Identity ("The Digital Atelier")
The site embodies intentional asymmetry, tonal depth, and editorial restraint:
- No border lines for sectioning — background color shifts create hierarchy
- 0px border radius throughout — sharp corners signal precision
- Forest Green (`#4b6450`) as a sparingly used gemstone accent
- Deep nocturnal palette anchored by `primary` black (`#000000`)
